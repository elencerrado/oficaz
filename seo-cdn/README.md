# Oficaz SEO Files CDN

This directory contains SEO files that should be hosted on a CDN for proper content-type serving.

## Setup Instructions:

1. Create a new GitHub repository (e.g., `oficaz-seo-cdn`)
2. Upload these files to the repository
3. Enable GitHub Pages for the repository
4. Configure redirects in your main application

## Files:
- `robots.txt` - Robots exclusion file
- `sitemap.xml` - XML sitemap

## CDN URLs (replace with actual):
- https://your-github-username.github.io/oficaz-seo-cdn/robots.txt
- https://your-github-username.github.io/oficaz-seo-cdn/sitemap.xml
